# Trainee SWE Screening Task - Sugarclass Limited

**Role:** Trainee Software Engineer

**Submission Deadline:** 4th April 2026, 8:00 PM

## Objective

Build a CPU-based PDF to Markdown converter. The task evaluates your ability to process files, structure logic, and document your work.

## Core Requirements

1. Accept a PDF file as input
2. Extract content
3. Convert into structured Markdown (.md)

## Expected Markdown Output

1. Headings (#, ##)
2. Paragraphs
3. Lists
4. Basic formatting (bold, italic)
5. Tables (bonus)
6. Code blocks (bonus)

## Technical Constraints

1. CPU only (no GPU)
2. Any language (Python preferred)
3. No paid APIs
4. Open-source libraries allowed

## Application Interface

1. CLI Tool (e.g., python app.py input.pdf output.md)
2. OR Minimal Web App (upload + download)

## Submission Requirements

1. Public GitHub repository
2. Clean source code
3. Detailed README (setup, usage, explanation, limitations)
4. Sample PDFs and outputs

## Evaluation Criteria

| Criteria | Weight |
| --- | --- |
| Code Quality | 25% |
| Accuracy | 25% |
| Approach | 20% |
| Documentation | 20% |
| Bonus | 10% |

## Bonus Points

1. Table extraction
2. OCR fallback
3. Smart formatting
4. Lightweight LLM cleanup

Focus on clarity, clean structure, and problem-solving approach over perfection.
